# Backend - Team ASAH A25-CS060

Backend API project capstone.

## Prerequisites

- Node.js (v22 atau lebih baru)
- npm

## Installation

```bash
npm install
```

## Environment Variables

Buat file `.env` di root folder:

```env
PORT=3000
HOST=0.0.0.0
NODE_ENV=development
```

Untuk production di Render, set environment variables di dashboard Render.

## Scripts

```bash
# Development dengan auto-reload
npm run dev

# Production
npm start

# Lint code
npm run lint

# Lint & auto-fix
npm run lint:fix
```
